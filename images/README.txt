BLUE SALON — IMAGE SLOTS
========================

Drop your real images into this folder using the exact filenames below.
Until a file exists, that slot shows a striped placeholder with its path.
No code changes needed — just match the filename.

Recommended: wide/landscape JPGs for banners; portrait JPGs for product
and category cards; transparent PNGs for brand logos.
